File Server API

Version v1.1

GET /

  • Returns quick API Guide

POST /login

  • Body:
    {
     username: string,
     password: string
    }
  • Returns a bearer token to put in 'Authorization' header
    • Authorization: 'Bearer [token]'

POST /createuser

  • Must be authenticated with admin privileges.
  • Creates a user with specified permissions.
    • Body:
      {
       username: string,
       password: string,
       permission: 'user' | 'admin'
      }

GET /file/?name=str&owner&str|undefined

  • Must be authenticated.
  • Returns file by the name of the filename

POST /file/?name=str&owner&str|undefined

  • Must be authenticated.
  • Writes file by name of the filename
